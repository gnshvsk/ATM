# Банкомат
Гонашевская Зоя, гр. ИВТ-173
Приложение "Банкомат" позволяет проводить операции с картами, добавленными в "базу данных", а именно в хэш-таблицу (использование класс HashSet).
Каждая карта имеет 2 уникальных параметра (номер и пин) и 1 не уникальный (баланс) 
Пользователь имеет возможность узнать баланс карты, а также снять определенную сумму денег. 
При попытки снятия суммы, которая больше баланса, обрабатывается исключение и появляется надпись "На вашем счету недостаточно средств"
